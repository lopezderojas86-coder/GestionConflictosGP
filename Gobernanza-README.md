
# Gobernanza

## Temas principales

- Gobernanza institucional
- Gobernabilidad democrática
- Capacidad institucional
- Confianza ciudadana
- Legitimidad institucional
- Articulación institucional
- Gestión colaborativa
- Liderazgo organizacional

## Relación con la tesis

La gobernanza busca mejorar la coordinación, participación y capacidad institucional para resolver problemas públicos de manera eficiente y transparente.
